Ключевые Навыки

HTML5

HTML («язык гипертекстовой разметки») - это код, который используется для структурирования и отображения веб-страницы и её контента.

CSS3

CSS («каскадные таблицы стилей») — это код, который используете для стилизации веб-страницы.

Javascript (es6+)

JavaScript — это язык программирования, который в первую очередь применяют в вебе. С его помощью сайты делают интерактивными и динамическими.

React JS

React - это JavaScript-библиотека для разработки пользовательских интерфейсов (в основном одностраничных и мобильных приложений).

Typescript

Typescript - это строго типизированный и компилируемый в JavaScript язык для выявления возможных ошибок, связанных с типами, на этапе компиляции.

AJAX

AJAX - это подход к построению интерактивных пользовательских интерфейсов веб-приложений, заключающийся в «фоновом» обмене данных с сервером.

Webpack

Webpack - это сборщик модулей для JavaScript-приложений, предназначенный для структурирования кода, оптимизации производительности и поддержки сторонних библиотек.

Адаптивная вёрстка

Адаптивная верстка — это способ создания веб-страниц, при котором они автоматически подстраиваются под размеры и ориентацию экрана устройства.

Sass

Sass - это препроцессор, который помогает сделать CSS-код понятнее и проще для дальнейшей поддержки, а также расширить его функциональность.

БЭМ

БЭМ - это компонентный подход к веб-разработке с разделением интерфейса на независимые блоки для повторного использования существующего кода.

Git

Git — это система контроля версий разработки, которая используется для отслеживания и ведения истории изменения файлов проекта.

Список экзотических фруктов

Используемые технологии:

  • HTML
  • CSS
  • Javascript

Учебный проект на JavaScript состоит из одной страницы. Основной акцент делался на изучение основ JavaScript, а также алгоритмов сортировки ('Быстрая' и 'Пузырьком')

Нажмите кнопку 'Перемешать', чтобы перемешать элементы списка в случайном порядке.

Нажмите кнопку 'Фильтровать', чтобы отфильтровать элементы списка по полю вес.

Нажмите кнопку 'Сортировать', чтобы отсортировать элементы списка по полю цвет в указанном порядком.

Нажмите кнопку 'Сменить алгоритм сортировки' для смены алгоритма сортировки по полю цвет. Доступны алгоритмы: bubbleSort и quickSort.

Нажмите кнопку 'Добавить фрукт' для добавления нового фрукта. Предварительно нужно обязательно заполнить все 3 поля (Вид, Цвет, Вес). Если их не заполнить, пользователю покажется сообщение с предупреждением об ошибке ввода.

Особенности проекта:

  • На странице реализована плавная анимация при взаимодействии с определенными элементами.
  • При определенных сценариях пользователь получает всплывающие сообщения (модальные окна).
  • Адаптирован под мобильные устройства.
